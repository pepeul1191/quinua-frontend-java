## Boilerplate Spark-Velocity-Hibernate

Requisitos de software previamente instalado:

	+ Java 1.8
	+ SBT
	+ NodeJS - NPM - Gulp

Instalación de dependencias:

	$ sbt eclipse && npm install && bower install
	$ sbt
		> run 

### Generar 'dist'
	
	$ gulp layout && gulp app && gulp error-css
